# Ограничения и roadmap

## Текущие ограничения

- Проверка ссылок на attachments/references основана на ключевых словах и может требовать уточнений.
- Сложные условные правила RI реализуются в табличной форме и требуют дисциплины заполнения.
- Без серверной части нет централизованной блокировки одновременного редактирования.

## Направления развития

- Добавить цифровую подпись и контроль неизменности после выпуска.
- Добавить расширенный diff ревизий DOCX.
- Добавить пакетный режим проверки нескольких документов.
- Добавить шаблоны для новых типов документов без изменения ядра.
